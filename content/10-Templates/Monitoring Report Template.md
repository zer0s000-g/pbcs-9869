# Monitoring Report Template

Copy this template into a working monitoring file and adapt it to the applicable State/regional monitoring programme.

## Source-routed monitoring report structure
| Period | Operation / capability | Metric | Source route | Required value | Actual value | Subset | Finding | Corrective action |
|---|---|---|---|---|---|---|---|---|
| YYYY-MM | CPDLC | ACP / ACTP / PORT | Appendix D Table D-1; [[Appendix D and E Monitoring Table Verification]] | applicable RCP | measured | operator / aircraft / CSP / media | | |
| YYYY-MM | ADS-C | ASP / transit time | Appendix D Table D-3; [[Appendix D and E Monitoring Table Verification]] | applicable RSP | measured | operator / aircraft / CSP / media | | |
| YYYY-MM | CPDLC/ADS-C availability | outage notification / outage duration | Appendix D section 2.3 | applicable availability criterion | measured | CSP / service / area | | |
| YYYY-MM | SATVOICE clearance | ACP | Appendix E Table E-1 | applicable RCP | measured | operator / aircraft / media | | |
| YYYY-MM | SATVOICE position report | ASP / delivery time | Appendix E Table E-2 | applicable RSP | measured | operator / aircraft / media | | |
| YYYY-MM | SATVOICE availability | outage notification / outage duration | Appendix E section 2.3 | applicable availability criterion | measured | CSP / service / area | | |

## Minimum field anchors
- CPDLC transaction fields: Appendix D, Table D-1, Doc 9869 PDF pages 154-156.
- ADS-C report fields: Appendix D, Table D-3, Doc 9869 PDF pages 159-160.
- SATVOICE clearance fields: Appendix E, Table E-1, Doc 9869 PDF page 198.
- SATVOICE position report fields: Appendix E, Table E-2, Doc 9869 PDF page 200.

## Boundary
Use this template to structure reports. Do not insert sample values from Doc 9869 as if they are company observed performance.
