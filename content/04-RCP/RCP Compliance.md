# RCP Compliance

## Summary
RCP compliance means showing that the operation, aircraft/operator, ANSP, CSP, and aircraft system meet the applicable RCP criteria and continue to meet them.

## Source basis
- Doc 9869 uses RCP specifications to define communication performance expectations for ATM operations.
- Appendix B is the main table family for RCP specification details; table-level checking is tracked in [[Appendix B RCP Table Verification]].
- For related implementation and monitoring context, see [[PBCS Implementation Lifecycle]] and [[RCP Monitoring]].

## Implementation questions
- Which ATM operation uses this RCP requirement?
- Which components receive allocated requirements?
- What evidence proves initial compliance?
- What monitoring detects continuing compliance or degradation?
