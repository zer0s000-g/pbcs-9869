# RSP Allocation

## Summary
RSP allocations distribute surveillance data delivery time, continuity, availability, integrity, monitoring/alerting, and safety requirements across system components.

## Source basis
- Doc 9869 uses RSP specifications to define surveillance performance expectations for ATM operations.
- Appendix C is the main table family for RSP specification details; table-level checking is tracked in [[Appendix C RSP Table Verification]].
- For related implementation and monitoring context, see [[PBCS Implementation Lifecycle]] and [[RSP Monitoring]].
