# Communication System Chain

## Summary
The communication chain includes controller/ATS systems, communication services/networks, aircraft systems, and flight crew response paths.

## Source basis
- Doc 9869 treats PBCS performance as an end-to-end operational system issue, not as a single equipment issue.
- This note explains how the relevant aircraft, ATS unit, communication, surveillance, and service-provider elements contribute to the PBCS chain.
- For role ownership and evidence routing, see [[PBCS Responsibility Matrix]] and [[Evidence and Traceability Map]].
