# RCP vs RSP

| Dimension | RCP | RSP |
|---|---|---|
| Main capability | Communication | Surveillance |
| Core performance time | Communication transaction time | Surveillance data delivery time |
| Monitored actual performance | ACP | ASP |
| Typical applications | CPDLC, SATVOICE communication transactions | ADS-C, position/report delivery, SATVOICE reporting context |
| Specification branch | [[RCP Specification]] | [[RSP Specification]] |
| Main appendix | Appendix B | Appendix C |

## Key takeaway
RCP and RSP are companion specifications. RCP asks whether the communication transaction supports the ATM operation; RSP asks whether surveillance information is delivered with required timeliness, continuity, availability, and integrity.
