# PBCS Responsibility Matrix

## Source posture
This matrix is a company-facing synthesis. It is source-hardened to Doc 9869 routing pages, but it is not a substitute for a State/regional rule, AIP entry, operational approval, or contract.

## Responsibility matrix
| Actor | Main responsibility in PBCS | Source route | Evidence examples |
|---|---|---|---|
| State authority | Set policy/objectives, prescribe or accept RCP/RSP requirements, and oversee approvals/compliance | Chapter 4, sections 4.1-4.3, Doc 9869 PDF pages 71-76 | regulations, approval policy, AIP/SUPP route applicability |
| ANSP | Implement the ATM operation, prescribe/consume RCP/RSP criteria operationally, run monitoring, coordinate corrective action | Chapter 3 sections 3.2-3.4; Chapter 4 section 4.5; Appendices D/E | implementation plan, ATS procedure/system evidence, monitoring reports |
| Aircraft operator | Demonstrate aircraft/operator capability, procedures, training, service arrangements, flight-plan use, and continued compliance | Chapter 4 sections 4.3-4.4; Appendices B/C component tables | operational approval, procedures, training, ORT/MEL evidence |
| CSP | Provide contracted communication service performance, outage notification, and support investigation/corrective action | Appendix B component tables; Appendix D/E availability sections | service agreement, outage logs, performance reports |
| SSP | Provide surveillance-related service performance and support monitoring/corrective action where applicable | Appendix C component tables; Appendix D/E monitoring sections | service agreement, surveillance delivery/performance reports |
| Aircraft manufacturer / avionics supplier | Provide aircraft system design, interoperability, and capability evidence used in approval/compliance | Appendix B/C aircraft system tables; referenced RTCA/EUROCAE standards | type design evidence, avionics compliance data, interoperability statement |

## Claim boundaries
- Doc 9869 supports role routing and evidence categories.
- It does not by itself prove that a specific State has prescribed a capability, that a specific aircraft/operator is approved, or that a specific provider currently meets contracted performance.
- Company implementation should add a private evidence layer for the actual operation.
