## Minimal mental model
PBCS is the performance-based way to specify, approve, and monitor the communication and surveillance capabilities that support an ATM operation.

RCP covers communication performance. RSP covers surveillance performance.

Doc 9869 links these specifications to State responsibilities, ANSP implementation, operator approval, service-provider arrangements, and post-implementation monitoring.
