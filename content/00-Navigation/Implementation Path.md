# Implementation Path

## Sequence
1. Identify the ATM operation and operational need: [[Developing RCP RSP Specifications]].
2. Select or prescribe applicable RCP/RSP criteria: [[Applying RCP RSP Specifications]].
3. Define stakeholder responsibilities: [[PBCS Responsibility Matrix]].
4. Establish approval and compliance evidence: [[Operational Approval]], [[PBCS Evidence Ladder]].
5. Implement monitoring: [[Post-Implementation Monitoring]].
6. Track non-compliance and recovery: [[Corrective Action]].
