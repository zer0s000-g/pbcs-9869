# How to Use This Knowledge Base

> [!summary] TL;DR
> This site works differently from a regular website. It's a knowledge graph — every page links to others, and you can navigate visually. This page shows you how. Takes **3 minutes**.

---

## Three ways to navigate

### 1. Linked reading (like a book)

Every page has wikilinks (underlined, colored text). Click any link to jump to that page.

The learning paths are designed for linear reading:
- [[Beginner Path]] → [[Implementation Path]] → [[Compliance Path]]
- Each page has a ⬅ ➡ navigation footer

This is the best way if you're learning PBCS for the first time.

### 2. The graph view (visual navigation)

In the right sidebar, you'll see "Graph View" — a network of connected dots.

- Each **dot** is a page
- Each **line** is a link between pages
- **Click any dot** to jump to that page
- Hover over a dot to see the page title
- The current page's dot is highlighted

> [!tip] Pro tip
> The graph view shows you how concepts relate. If two pages are densely connected, they're closely related topics. It's a faster way to discover content than clicking through menus.

### 3. Search (find anything instantly)

Click the **Search** button (top of the left sidebar) or press `Ctrl+K` / `Cmd+K`.

Type any word — "CPDLC", "transaction time", "monitoring", "Appendix B" — and results appear instantly. Search covers every word on every page.

> [!tip] Another tip
> Unlike a regular search engine, Quartz search is **full-text and real-time**. You don't need to know the exact page title — type any phrase from any page.

## Key features explained

### Explorer (left sidebar)

The folder tree on the left shows the page hierarchy. Click any folder to expand it. Click a page to navigate.

The page you're currently viewing is highlighted.

### Backlinks

At the bottom of the right sidebar, "Backlinks" shows every page that links to the current page.

This answers: *"What other pages reference this concept?"* It's how you discover related content you might have missed.

### Table of Contents

For longer pages, the right sidebar shows a "Table of Contents" with all headings. Click any heading to jump to that section.

### Dark mode / Reader mode

- **Dark mode** button (moon/sun icon) — toggle light/dark theme
- **Reader mode** button (book icon) — temporarily hide sidebars for focused reading

## Site conventions

### Callout boxes

You'll see colored boxes throughout the site:

| Color | Meaning |
|---|---|
| 🔵 Blue (`note`, `info`) | Background information |
| 🟢 Green (`tip`, `example`) | Helpful hints and worked scenarios |
| 🟡 Yellow (`warning`, `caution`) | Things to watch out for |
| 🔴 Red (`danger`) | Critical warnings |

### Verification badges

Pages carry a `verification_status` indicating how thoroughly they've been checked against Doc 9869:

| Badge | Meaning |
|---|---|
| `source-linked` | Concept traced to a Doc 9869 section, but not word-for-word verified |
| `source-hardened-direct-pdf-check` | Cross-checked against the actual PDF text |

### Acronyms

PBCS is full of acronyms. If you encounter an unfamiliar one, check the [[Acronyms and Terms]] page — it covers all of them with definitions and links.
