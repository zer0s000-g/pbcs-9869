# Aircraft Operator Checklist

## Summary
Operator checklist for aircraft capability, operational approval, procedures, training, flight planning, CSP/SSP arrangements, and monitoring follow-up.

## Source basis
- Doc 9869 describes PBCS implementation as a lifecycle: selecting the applicable RCP/RSP specifications, showing compliance, obtaining operational approval where required, and monitoring continued performance.
- This note turns that source structure into practical implementation guidance.
- For evidence routing, see [[PBCS Compliance Matrix Template]] and [[Evidence and Traceability Map]].

## Practical outputs
- Named owner.
- Required evidence.
- Monitoring data source.
- Review cadence.
- Escalation/corrective-action path.
