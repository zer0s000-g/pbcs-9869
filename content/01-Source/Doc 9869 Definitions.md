# Doc 9869 Definitions

This note centralizes high-value PBCS terms used across the knowledge base. It is written as a working glossary for colleagues. For legal or formal compliance wording, verify the term in the original Doc 9869 glossary and the applicable State or regional material.

## Performance and monitoring terms

| Term | Working meaning in this knowledge base | Source basis |
|---|---|---|
| Actual communication performance (ACP) | The observed communication performance measured from operational data and compared with the applicable RCP monitored-performance value. | Doc 9869 glossary and Appendix D/E monitoring material |
| Actual surveillance performance (ASP) | The observed surveillance delivery performance measured from operational data and compared with the applicable RSP monitored-performance value. | Doc 9869 glossary and Appendix D/E monitoring material |
| Communication transaction time | The time used to assess whether a communication transaction completed quickly enough for the intended ATM operation. | Doc 9869 glossary, Chapter 1, Appendix B, Appendix D |
| Surveillance data delivery time | The time used to assess whether surveillance data was delivered quickly enough for the intended ATM operation. | Doc 9869 glossary, Chapter 1, Appendix C, Appendix D |
| Continuity | The probability that the communication or surveillance function will complete as required when it is needed. | Doc 9869 Chapter 2 and Appendix B/C tables |
| Availability | The probability that the required communication or surveillance service is available when required for the operation. | Doc 9869 Chapter 2 and Appendix B/C tables |
| Integrity | The protection against undetected or misleading communication/surveillance performance failures. | Doc 9869 Chapter 2 and Appendix B/C tables |

## RCP and RSP terms

| Term | Working meaning in this knowledge base | Source basis |
|---|---|---|
| Required communication performance (RCP) | The communication-performance requirement used to specify and assess communication capability for a defined ATM operation. | Doc 9869 Foreword, Chapter 1, Chapter 2 |
| RCP specification | A set of communication performance, functional, safety, allocation, and monitoring criteria used for implementation and compliance. | Doc 9869 Chapter 2 and Appendix B |
| Required surveillance performance (RSP) | The surveillance-performance requirement used to specify and assess surveillance capability for a defined ATM operation. | Doc 9869 Foreword, Chapter 1, Chapter 2 |
| RSP specification | A set of surveillance performance, functional, safety, allocation, and monitoring criteria used for implementation and compliance. | Doc 9869 Chapter 2 and Appendix C |
| RCP 240/D | A CPDLC-related RCP specification used for controller intervention capability in data-link operations. | [[RCP 240 D]] and [[Appendix B RCP Table Verification]] |
| RSP 180/D | An ADS-C-related RSP specification used for surveillance data delivery in data-link operations. | [[RSP 180 D]] and [[Appendix C RSP Table Verification]] |

## Services and technologies

| Term | Working meaning in this knowledge base | Source basis |
|---|---|---|
| CPDLC | Controller-pilot data link communications. In this KB, CPDLC is mainly associated with RCP specifications and communication transaction monitoring. | Doc 9869 Chapter 1, Appendix B, Appendix D |
| ADS-C | Automatic dependent surveillance — contract. In this KB, ADS-C is mainly associated with RSP specifications and surveillance delivery monitoring. | Doc 9869 Chapter 1, Appendix C, Appendix D |
| SATVOICE | Satellite voice communications. In this KB, SATVOICE is mainly associated with RCP/RSP VRO material and Appendix E monitoring. | Doc 9869 Chapter 1, Appendix B/C, Appendix E |

## Stakeholder terms

| Term | Working meaning in this knowledge base | Source basis |
|---|---|---|
| State authority | The authority responsible for policy, prescription, approval, oversight, and State-level compliance expectations. | Doc 9869 Chapter 3 and Chapter 4 |
| Air navigation service provider (ANSP) | The operational service provider responsible for applying requirements in ATM service delivery, procedures, monitoring, and corrective-action coordination. | Doc 9869 Chapter 3, Chapter 4, Appendix D/E |
| Aircraft operator | The operator responsible for aircraft/operator capability, procedures, training, approvals, flight planning, and continued compliance evidence. | Doc 9869 Chapter 4 and Appendix B/C component tables |
| Communication service provider (CSP) | The provider responsible for communication-service performance contributions and service availability evidence where applicable. | Doc 9869 Appendix B/D/E component and monitoring material |
| Surveillance service provider (SSP) | The provider responsible for surveillance-service performance contributions and service availability evidence where applicable. | Doc 9869 Appendix C/D/E component and monitoring material |
| Aircraft manufacturer / avionics supplier | The organizations that provide design, equipage, interoperability, and capability evidence used in approval and compliance. | Doc 9869 Appendix B/C component material and referenced technical standards |
